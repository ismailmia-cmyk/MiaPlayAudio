# MiaPlayAudio
Audio Player
MiaPlay Audio Player

A sleek, modern browser-based Audio player with playlist management capabilities. MiaPlay offers an intuitive interface for playing your favourite audio directly in the browser with no server requirements.

Features - Browser-Based Audio Player - Play MP3, WAV, OGG, M4A/AAC, WebM/Opus, and FLAC files directly in your browser

Playlist Management - Create and manage your music playlists

Drag & Drop Reordering - Easily rearrange songs in your playlist

Intuitive Controls - Play, pause, skip, shuffle, and repeat functionality

Responsive Design - Works seamlessly on desktop and mobile devices

Modern UI - Clean, dark theme with gradient accents

Volume Control - Adjustable audio levels

Progress Seeking - Click on the progress bar to jump to any point in the song

Easy Song Removal - Remove individual songs or clear entire playlist

Quick Start Download the files to your local machine

Open index.html in your web browser

Click "Add Audio Files" to upload your music

Start playing - Click on any song in the playlist

How to Use Adding Music Click the "+ Add Audio Files" button on the panel

Select one or multiple audio files from your computer

Songs will be added to your playlist automatically

Playlist Management Play a song: Click on any song in the playlist

Reorder songs: Drag songs using the handle (⋮⋮ icon) on the left of each track

Remove a song: Hover over a track and click the × button that appears

Clear playlist: Use the "Clear All" button in the playlist header

Player Controls Play/Pause: Centre button with play/pause icons

Previous/Next: Skip between tracks

Shuffle: Randomise playback order (indicator turns purple when active)

Repeat: Loop current track (indicator turns purple when active)

Volume: Use the slider to adjust audio levels

Progress: Click anywhere on the progress bar to seek

Technical Details Pure HTML, CSS, and JavaScript - No frameworks or dependencies required

File API - Uses the browser's File API for local file handling

Drag & Drop API - Native browser APIs for playlist reordering

Audio Element - Standard HTML5 audio element for playback

Responsive CSS - Flexbox and media queries for adaptive layout

Browser Compatibility. Browser Version Support: Chrome 60+ (Full Support), Firefox 55+ (Full Support), Safari 11+ (Full Support), and Edge 79+ (Full Support).
